# 《那年盛夏》PPT封面设计方案

## 方案一 · 天空辽阔版

```
┌─────────────────────────────────────────────┐
│                                             │
│          ☁️        ☁️              ☁️       │
│     ☁️                      ☁️              │
│                                             │
│          蓝 天 渐 变 背 景                   │
│                                             │
│                                             │
│              那 年 盛 夏                     │
│                                             │
│         —— 致我们一起走过的青春              │
│                                             │
│                                             │
│          有些夏天，注定被记住一辈子           │
│                                             │
│                                             │
│    ▫  高三( )班毕业纪念  ▫                   │
│                                             │
│  ───────── 校园操场剪影 ─────────           │
└─────────────────────────────────────────────┘
```

**设计要点：**
- 上70%为蓝天白云渐变背景
- 标题居中，大字"那年盛夏"用站酷小薇LOGO体或思源宋体
- 底部10%为校园/操场/树木剪影
- 副标题用细体字，间距拉开
- 左下角可加班级信息

**AI配图关键词（封面背景图）：**
```
anime wide blue sky with fluffy white clouds, school campus silhouettes at bottom,
summer atmosphere, cinematic graduation feeling, Makoto Shinkai style,
warm golden sunlight, panoramic composition, 16:9 aspect ratio
```

## 方案二 · 黄昏温暖版

```
┌─────────────────────────────────────────────┐
│                                             │
│   🌅 夕 阳 渐 变（橙 → 粉 → 紫 → 蓝）     │
│                                             │
│                                             │
│          那 年                              │
│          盛 夏                              │
│                                             │
│     "有些夏天，注定被记住一辈子"             │
│                                             │
│                                             │
│   校园剪影  │  操场跑道  │  教学楼剪影       │
│  ─────────────────────────────────────      │
└─────────────────────────────────────────────┘
```

**设计要点：**
- 背景从橙色到粉紫到深蓝的黄昏渐变
- 标题两行竖排，更有电影感
- 底部校园元素剪影（教学楼、操场、树木）
- 引用文案用浅色+透明度80%

**AI配图关键词：**
```
anime sunset sky gradient, orange pink purple blue, school building silhouettes,
graduation emotional atmosphere, cinematic wide shot, warm golden hour,
nostalgic anime style, beautiful dramatic sky, 16:9
```

## 方案三 · 极简电影海报版

```
┌─────────────────────────────────────────────┐
│                                             │
│                                             │
│                 蓝天白云照片                 │
│              （占满整个画面）                │
│                                             │
│                                             │
│                                             │
│    ───── 那 年 盛 夏 ─────                  │
│                                             │
│    致我们一起走过的青春                      │
│                                             │
│                                             │
└─────────────────────────────────────────────┘
```

**设计要点：**
- 全屏蓝天白云实景/AI图
- 中间加一条半透明白色横条
- 标题放在横条上，黑色或深蓝色字体
- 极简，类似电影海报
- 下方小字副标题

## Canva制作步骤

1. 新建1920×1080画布
2. 搜索"蓝天白云"背景图（或上传AI生成的图）
3. 添加标题文字"那年盛夏"，选择宋体类字体，60pt
4. 添加副标题，选择细黑体，24pt
5. 底部添加校园元素装饰
6. 左下角加班级信息
7. 导出为PNG/JPG
