# 小学数学互动课件（网页小游戏）

纯前端 HTML/CSS/JS，适合课堂或家庭练习。推送到 `main` 分支后由 GitHub Actions 自动部署到 GitHub Pages。

## 在线访问

部署成功后（仓库 **Settings → Pages** 已启用）：

| 课件 | 路径 |
|------|------|
| 站点首页 | `/` |
| 一年级 | `/` 或 `/数学一年级/` |
| 二年级 | `/grade2/` 或 `/数学二年级/` |
| 三年级 | `/grade3/` 或 `/数学三年级/` |
| 认识面积 | `/area.html` |

示例：`https://<你的用户名>.github.io/shuxue1/grade3/`

## 验证码设置

各年级在 `js/data.js` 顶部修改 `ACCESS_CONFIG`：

```javascript
const ACCESS_CONFIG = {
  code: "3366",           // 三年级默认 3366，二年级 2266，一年级 8866
  hint: "请输入老师或家长提供的验证码",
  rememberDays: 7,        // 勾选「7 天内免重复输入」的有效天数
};
```

- 验证通过后在浏览器 `localStorage` 保存令牌（非明文密码）
- 页头 🔒 可退出并清除登录状态
