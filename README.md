## 🐉 端午节小游戏

位于 `端午节/`，包含两个互动小游戏：

1. **包粽子工坊（自由创作）**：先选粽叶形状（三角粽 / 长粽 / 牛角粽，全 SVG 绘制），再从 15 种食材里**自由点选添加**（每种可放多份）—— 糯米 / 红枣 / 豆沙 / 蜜枣 / 花生 / 红豆 / 莲子 / 板栗 / 葡萄干 / 鲜肉 / 咸蛋黄 / 火腿 / 白糖 / 巧克力 / 辣椒；所有食材均为 SVG 绘制（不依赖 emoji，外形与真实食材一致）。包好后有麻绳一圈圈缠绕的 SVG 动画，下锅煮 → 出锅时根据食材组合**智能命名 + 评分**（甜度/咸度/创意/满足感四维），可命名为「红枣粽」「蛋黄鲜肉粽」「豪华八宝粽」「创意巧克力粽」「重口辣味粽」「清水白米粽」等 12 种。
2. **划龙舟竞渡（三档难度）**：
   - 新手村：10 格 · 25 秒/题 · 对手稳速
   - 进阶赛：12 格 · 15 秒/题 · 对手稳健
   - 大师赛：15 格 · 10 秒/题 · 对手越战越勇
   - 答对一题前进 1 格，**连对 3 题加速前进 2 格** 🚀；超时算错；30 题题库（端午文化 18 + 数学应用 12）按难度筛选；对手 AI 速率会随你的对错动态调整
   - 冠军 🏆（大师赛 +5 ⭐ / 进阶 +3 ⭐ / 新手 +2 ⭐）；安慰 🚩 +1 ⭐；平局 🏆🚩 +2 ⭐
