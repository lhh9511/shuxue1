## 验证码设置

各年级在 `js/data.js` 顶部修改 `ACCESS_CONFIG`：

```javascript
const ACCESS_CONFIG = {
  code: "6666",           // 六年级默认 6666，五年级 5566，四年级 4466，三年级 3366，二年级 2266，一年级 8866
  hint: "请输入老师或家长提供的验证码",
  rememberDays: 7,        // 勾选「7 天内免重复输入」的有效天数
};
```

- 验证通过后在浏览器 `localStorage` 保存令牌（非明文密码）
- 页头 🔒 可退出并清除登录状态

## 🐉 端午节小游戏（无需验证码）

位于 `端午节/`，包含两个互动小游戏：

1. **粽子工坊**：5 种口味（红枣 / 豆沙 / 蛋黄鲜肉 / 蜜枣花生 / 八宝）；点击 12 种食材"加进粽叶"；满足配方后"包好下锅"，出锅有撒花动画。
2. **划龙舟竞渡**：10 格水道双轨（你 vs 对手），答对一题前进 1 格，对手有 65% 概率前进；先到终点拿 🏆，落败拿 🚩。题库 16 题，融合端午文化与小学数学。
